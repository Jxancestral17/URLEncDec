# URLEncDec

<p>Simple URL encoder and decoder, written in go. xss url encoding utility</p>

```bash
git clone https://github.com/Jxancestral17/URLEncDec.git
```
Create a build

```bash
make build
```

Help 

```bash
./urlencdec -h
```


Encode

```bash
./urlencdec -e -i {name_file}
```

Decode

```bash
./urlencdec -d -i {name_file}
```
